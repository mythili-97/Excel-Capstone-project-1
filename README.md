# Data cleaning,analysis and visualization with excel
The aim of the project is to perform data cleaning with use of pivot tables for data visualization.
## Description
The project has an E-commerce dataset which has four tables which includes information about product,customer,store and sales of the products.Use of pivot charts and tables for deeper analysis and visualization for accuracy of the given data.


## Getting Started

### Dependencies
Micrososft Excel

### Installing
We can download the dataset and can be opened using Micrososft Excel.

### Executing program
We can calculate values using formulas for which output is verified
